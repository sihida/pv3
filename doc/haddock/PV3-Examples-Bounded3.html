PV3.Examples.Bounded3

Description

Example that squares the first (int) parameter, using two loops. Combination of Bounded and Bounded2.

Pseudo-Java:

int p (int a0) { var x0 = 0; var x1 = 0; x1 = a0; while (x1 > 0) { x0 = x0 + a0; x1 = x1 - 1; } x1 = 0; var x2 = 0; while (x1 < 1) { x2 = x0 + x2; x1 = x1 + 1; } return x2; }

Pre: a0 >= 0 Post: return == a0*a0
